Something im noticing about the commands is all the extra arguments they
take - its sort of becoming a lot like using a DOS command :-/

Why is there a target in .conceal ? .setprop concealed # works. Remember -
conceal can be any number.
If mob.concealed is greater than your command level, you can no longer see
it.

Also, how do I unconceal when I am ingame other than .setprop concealed 0 on
myself?

I think a better script would be ".conceal <level>"

program textcmd_Conceal(who, number)
number := CInt(number);
if ( number > who.cmdlevel )
number := who.cmdlevel;
SendSysMessage(who, "Conceal level too high; resetting to " + number + ".");
endif

SendSysMessage(who, "Concealed:"+number);
who.concealed := number;
endprogram
